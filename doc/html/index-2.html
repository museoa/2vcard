2. Installation

2.1 How to obtain 2vcard

2vcard 0.1 can be obtained from:

  • http://www.netmeister.org/apps/2vcard-0.1.tar.gz - as a gzipped source-tarball
  • http://www.netmeister.org/apps/2vcard-0.1.tar.bz2 - as a bzip2'ed source-tarball
  • http://www.netmeister.org/apps/2vcard-0.1.i386.rpm - as a RPM
  • http://www.netmeister.org/apps/2vcard-0.1.src.rpm - as a source RPM
  • http://www.netmeister.org/apps/2vcard_0.1-1_i386.deb - as a devian package

2.2 Requirements

In order to successfully use 2vcard, you need the following programs and libraries which are available on most platforms as distribution packages and thereby can be installed easily.

Required:

  • Perl http://www.perl.com

2.3 Compilation and installation

2vcard assumes that the Perl executable lives in /usr/bin/perl - if that is not the case, use your favorite editor to modify the first line of the file 2vcard accordingly.

In order to install 2vcard from sources, type the following in the directory to which you downloaded the 2vcard distribution:

% tar zxvf 2vcard-0.1.tar.gz
% cd 2vcard-0.1
% less README
% su
% make install

This will install 2vcard in /usr/bin, the man page in /usr/share/man/man1/ and the documentation in /usr/share/doc/2vcard/

To install from RPM or .deb, please refer to rpm(1) or dpkg(1).

Should you run into problems please report them to the the author at jschauma@netmeister.org
